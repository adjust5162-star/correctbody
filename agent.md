# Agent Workflow

## 총괄 관리자

프리미엄 재활운동 랜딩페이지라는 목표를 기준으로 기획, 디자인, 개발, 디버깅을 조율하고 최종 실행 가능 여부를 확인합니다.

## 기획자

방문자가 문제 공감, 재활운동 필요성 이해, 프로그램 탐색, 전문가 신뢰 확인, FAQ 확인, 상담 문의로 이동하도록 섹션 흐름을 설계했습니다.

## 디자이너

넓은 여백, 큰 타이포그래피, 연한 블루 그라데이션, 고급 카드 레이아웃, 명확한 CTA를 사용했습니다. 특정 Dribbble/Awwwards 디자인을 복제하지 않고 감성만 재해석했습니다.

## 개발자

Next.js App Router, React, TypeScript, Tailwind CSS, lucide-react 아이콘으로 실제 실행 가능한 랜딩페이지를 구현했습니다.

## 디버깅 담당자

다음 항목을 검증합니다.

- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run build`
- Tailwind CSS 적용
- 모바일 반응형
- 상담 폼 검증과 제출
- 버튼 및 앵커 링크
- 콘솔 에러
- 의료 과장 표현
