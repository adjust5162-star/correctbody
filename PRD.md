# PRD

## 목표

Dribbble과 Awwwards에서 볼 수 있는 고급 랜딩페이지의 장점인 세련된 레이아웃, 넓은 여백, 강한 첫인상, 명확한 CTA 구조를 재활운동 서비스에 맞게 재해석합니다. 특정 디자인을 복제하지 않고, 프리미엄 재활운동 브랜드처럼 보이도록 구성합니다.

## 핵심 사용자

- 목, 어깨, 허리, 무릎 통증을 겪는 사용자
- 수술 후 재활운동 방향을 찾는 사용자
- 거북목, 라운드 숄더, 체형 교정 운동을 찾는 사용자
- 시니어 낙상 예방 운동을 알아보는 사용자
- 상담 또는 예약 문의를 원하는 사용자

## 랜딩 흐름

1. 첫 화면에서 문제 공감과 신뢰 형성
2. 통증 반복의 원인을 움직임 관점으로 설명
3. 재활운동과 일반 운동의 차이 제시
4. 대표 프로그램을 카드로 안내
5. 증상별 콘텐츠 탐색 유도
6. 전문가 철학과 접근 방식 제시
7. 상담부터 운동 진행까지 과정 안내
8. FAQ로 불안 해소
9. 상담 문의 CTA로 전환

## 필수 기능

- 랜딩페이지 앵커 내비게이션
- 모바일 햄버거 메뉴
- 상담 폼 필수값 검증
- 상담 API 서버 검증
- SEO 메타데이터
- 의료 주의 문구 포함
- 의료 과장 표현 배제

## SEO

메타 타이틀은 `재활운동 전문 랜딩페이지 | 통증 회복과 맞춤 운동`입니다. 주요 키워드는 재활운동, 통증 회복 운동, 허리 통증 운동, 무릎 통증 재활, 어깨 통증 스트레칭, 거북목 교정운동, 라운드 숄더 운동, 수술 후 재활운동, 시니어 낙상 예방 운동, 물리치료 운동입니다.
