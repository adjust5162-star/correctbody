# Design System

## 방향

Premium, Clean, Medical, Warm, Modern, Trustworthy, Motion, Recovery를 핵심 키워드로 삼았습니다. 병원식 신뢰감과 운동 브랜드의 활기를 함께 유지합니다.

## 컬러

- Primary: `#2563EB`
- Secondary: `#10B981`
- Accent: `#F97316`
- Background: `#FFFFFF`, `#F8FAFC`, `#EEF6FF`
- Text: `#111827`, `#374151`, `#6B7280`

## 타이포그래피

- H1: 데스크톱 56px 내외, 모바일 34px 내외
- H2: 데스크톱 40px 내외, 모바일 28px 내외
- Body: 16px-18px, 넉넉한 줄간격
- Font: Geist, Noto Sans KR, system sans-serif

## UI 규칙

- Hero는 넓은 화면과 강한 카피를 사용
- CTA는 `상담 문의하기`, `프로그램 보기` 두 갈래로 제공
- 섹션 배경은 흰색과 연한 블루/그레이를 교차
- 카드는 8px radius, 약한 그림자, hover lift 적용
- 버튼은 48px 이상, 모바일 터치 친화적 크기
- 과한 인터랙션 대신 fade-up과 hover 정도의 부드러운 모션 사용

## 접근성

- 모든 주요 이미지는 alt 텍스트 제공
- 버튼과 링크는 focus ring 제공
- 모바일 메뉴는 aria-label과 aria-expanded 제공
- 의료 정보는 보장 표현 없이 안내성 문장으로 작성
